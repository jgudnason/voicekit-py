# voicekit-py

A voice analysis toolbox in Python, implemented from first principles:
glottal inverse filtering, glottal closure/opening instant (GCI/GOI)
detection, and voice source feature extraction.

See [DESIGN.md](DESIGN.md) for scope, architecture, and roadmap.

## Status

Early development. Not yet installable or usable.
